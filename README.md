# Maravilhoso_docs

Projeto apresentado na formação interna da DASA no Forma Tech 2022.

*Tecnologias*:

- [x] React Native
- [x] Styled Commponents

- [ ] GraphQL

*Screens*:

- [x] SignUp (Validação de inputs e toast de "salvo com sucesso")
- [x] Uma lista
